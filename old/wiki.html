Что это? - это бот BlackSmith [Neutron // Talisman // Jaskier] (old)

Зачем он тебе сдался? - это один из лучших ботов на оси Neutron (ИМХО)

Как установить:

!)  Необходимо иметь интерпритатор python не ниже версии 2.5.2 (Python 3 isn't suppoted!)

Скачайте бота:

Для Unix (исходник):
->  svn checkout http://blacksmith-bot.googlecode.com/svn/old BlackSmith
или
->  svn checkout https://mysvn.ru/jabberik/blacksmith/trunk BlackSmith

Для Windows (exe installer):
->  http://blacksmith-bot.google.../BlackSetup.exe

Или-же с официального сайта проекта -> http://witcher-team.ucoz.ru/load/

Запуск:

>>> Unix:
1)  Редактируте sourse.py (BlackSmith/static)
    !) Предварительно переименуйте sourse.ru или source.en в sourse.py
2)  Командуем в консоли "python путь_кдиректории_бота/BlackSmith/BlackSmith.py" (или starter.py)
    !) Под стартером выключение невозможно без предварительного отключения стартера (kill -9 pid_стартера)

>>> Windows:
1)  Запускаем BlackSetup.exe и устанавливаем бота, заполняем открыйвшийся файл конфигурации.
2)  Путём банального дабл-клика по ярлыку BlackSmith или BlackStarter запускаем бота.

3)  Из ростера с жида администратора командуйте "джойн chat[at]conference.tld"

-> И вауаля! Бот обслуживает вашу конференцию!
